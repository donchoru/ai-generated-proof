# Claude Code 기술 아키텍처 — 외부 정보 주입 불가 증명

## Claude Code 개요

Claude Code는 Anthropic이 개발한 공식 CLI(Command Line Interface) 코딩 에이전트입니다.
터미널에서 실행되며, 사용자의 자연어 지시를 받아 코드와 문서를 생성합니다.

- **공식 문서**: https://docs.anthropic.com/en/docs/claude-code
- **개발사**: Anthropic (미국 AI 안전 기업)
- **모델**: Claude Opus 4.6 (2025년 사전 학습 완료)

## 핵심 보안 특성

### 1. 입력 경로 제한

Claude Code는 터미널 CLI입니다. 사용자가 정보를 입력하는 유일한 방법은 **키보드로 텍스트를 타이핑**하는 것입니다.

- 파일 드래그앤드롭: **불가**
- 클립보드 자동 붙여넣기: **불가** (사용자가 의도적으로 Ctrl+V/Cmd+V로 붙여넣어야 함)
- 외부 API 자동 연결: **불가** (사용자가 명시적으로 지시하지 않는 한)

따라서 사내 시스템의 데이터가 "자동으로" Claude에 유입되는 것은 구조적으로 불가능합니다.

### 2. 완전한 감사 추적 (Audit Trail)

Claude Code는 모든 상호작용을 JSONL 형식의 세션 로그에 기록합니다:

```
~/.claude/projects/<project-hash>/<session-id>.jsonl
```

각 기록에는:
- **타임스탬프** (UTC)
- **메시지 유형** (human = 사용자, assistant = AI)
- **전체 내용** (사용자가 입력한 텍스트 전문, AI가 생성한 텍스트 전문)
- **도구 호출** (Write, Edit, Bash 등 — 파라미터 포함)
- **모델 ID** (예: `claude-opus-4-6`)

이 로그를 검사하면 사용자가 입력한 모든 내용을 확인할 수 있으며,
사내 정보를 입력한 기록이 있는지 없는지를 완벽하게 검증할 수 있습니다.

### 3. AI 모델의 지식 범위

Claude Opus 4.6의 학습 데이터:
- **범위**: 2025년 5월 이전의 공개된 인터넷 데이터
- **포함**: 위키피디아, 교과서, 기술 문서, 공개 논문, 뉴스 기사 등
- **미포함**: 비공개 기업 데이터, 사내 문서, 유료 데이터베이스

반도체 공정에 대한 AI의 지식은 공개된 교과서/논문 수준이며,
특정 회사의 내부 프로세스, 장비 세팅, 생산 데이터 등은 포함되지 않습니다.

### 4. 대화 데이터 학습 미사용

Anthropic의 정책에 따라, Claude Code를 통한 사용자와의 대화는 **모델 학습에 사용되지 않습니다**.
사용자가 입력한 내용이 다른 사용자에게 노출될 위험도 없습니다.

---

## 본 사안에의 적용

1. 사용자는 "반도체 공정 매뉴얼 만들어줘" 등의 **주제 키워드만 입력**
2. Claude는 사전 학습된 **일반적인 반도체 공정 지식**으로 텍스트를 생성
3. 생성된 내용은 교과서에서 찾을 수 있는 수준의 일반 정보
4. 세션 로그로 사용자 입력 전체를 검증 가능 → 사내 정보 입력 기록 없음
5. Claude Code의 구조상, 사용자가 명시적으로 타이핑하지 않은 정보는 AI에 전달될 수 없음
